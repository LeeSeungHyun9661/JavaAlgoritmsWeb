Radix Sort

Radix Sort is a non-comparison sorting algorithm that sorts integers by processing individual digits of the numbers. It works by distributing the elements into buckets based on their digits, then collecting the elements from the buckets in a specific order.

History

Radix Sort was first proposed by Herman Hollerith in 1887 for use with punched card machines. It became popular in the computer science community for its linear time complexity.

Working Principle and Example

Consider an array to be sorted: [170, 45, 75, 90, 802, 24, 2, 66].

Step.1
  • Sort the elements based on their least significant digit (units place)
  • After this step, the array becomes: [170, 90, 802, 2, 24, 45, 75, 66]
Step.2
  • Sort the elements again based on their next significant digit (tens place)
  • After this step, the array becomes: [802, 2, 24, 45, 66, 170, 75, 90]
Step.3
  • Repeat the sorting process for all significant digits until the most significant digit is reached
  • After the final step, the array is sorted: [2, 24, 45, 66, 75, 90, 170, 802]
Result

[2, 24, 45, 66, 75, 90, 170, 802] (sorted)

Time Complexity

  • Time complexity: \(O(d \cdot (n + k))\), where \(d\) is the number of digits, \(n\) is the number of elements, and \(k\) is the base of the number system (typically 10 for decimal numbers)

Summary

Radix Sort is a stable sorting algorithm that is especially effective for sorting integers with a fixed number of digits. It achieves linear time complexity by leveraging the radix-based sorting technique.

Java code

        public void sort() {
            int[] list = super.getList();
            int topRadix = getTopRadix(list);
    
            Queue[] buckets = new LinkedList[10];
            for (int i = 0; i < 10; i++) {
                buckets[i] = new LinkedList<>();
            }
    
            int radix = 0;
            while (topRadix > radix) {
                int listIdx = 0;
    
                for (int i : list) {
                    int num = (int) (i / Math.pow(10, radix) % 10);
                    buckets[num].add(i);
                }
    
                for (int j = 0; j < 10; j++) {
                    while (!buckets[j].isEmpty()) {
                        list[listIdx++] = buckets[j].poll();
                    }
                }
                radix++;
            }
    
        }
                    
        public static int getTopRadix(int[] list) {
            int top = 0;
            for (int i : list) {
                if (i > top)
                    top = i;
            }
            int radix = 0;
            while (top > 0) {
                top /= 10;
                radix++;
            }
            return radix;
        }
