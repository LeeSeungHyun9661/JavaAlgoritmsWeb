Insertion Sort

Insertion Sort is a simple and efficient comparison-based sorting algorithm. It builds the final sorted array one item at a time, with the movement of elements being akin to sorting playing cards in your hands.

History

Insertion Sort has been used since the early days of computing and is still widely taught due to its simplicity and ease of understanding. It is efficient for small datasets or partially sorted arrays and serves as an introduction to more complex algorithms.

Working Principle

Insertion Sort iterates through the list, consuming one input element per iteration and growing a sorted output list. At each iteration, Insertion Sort removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there. It repeats until no input elements remain. Insertion Sort is an in-place sorting algorithm with a space complexity of \(O(1)\). It is also a stable sorting algorithm, maintaining the relative order of equal elements.

Consider an array to be sorted: [5, 3, 8, 4, 2].

Step.1
  • [5, 3, 8, 4, 2] (Initial array)
  • 3 is compared with 5 and inserted before it → [3, 5, 8, 4, 2]
Step.2
  • [3, 5, 8, 4, 2]
  • 8 is compared with 5 and remains in place → [3, 5, 8, 4, 2]
Step.3
  • [3, 5, 8, 4, 2]
  • 4 is compared with 8 and inserted before it → [3, 5, 4, 8, 2]
  • 4 is then compared with 5 and inserted before it → [3, 4, 5, 8, 2]
Step.4
  • [3, 4, 5, 8, 2]
  • 2 is compared with 8 and inserted before it → [3, 4, 5, 2, 8]
  • 2 is then compared with 5 and inserted before it → [3, 4, 2, 5, 8]
  • 2 is then compared with 4 and inserted before it → [3, 2, 4, 5, 8]
  • 2 is finally compared with 3 and inserted before it → [2, 3, 4, 5, 8]
Result

[2, 3, 4, 5, 8] (sorted)

Time Complexity

  • Best-case time complexity: \(O(n)\) (when the array is already sorted)
  • Average and worst-case time complexity: \(O(n^2)\)

Summary

Insertion Sort is a simple and easy-to-understand algorithm that works efficiently on small datasets or nearly sorted arrays. Java code

        public void sort() {
            int[] list = getList();
    
            for (int i = 1; i < list.length; i++) {
                int target = i;
                for (int j = i - 1; j >= 0; j--) {
                    if (list[target] < list[j]) {
                        swap(target--, j);
                    } else {
                        break;
                    }
                }
            }
        }
