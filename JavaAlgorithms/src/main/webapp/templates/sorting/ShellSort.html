Shell Sort

Shell Sort, also known as diminishing increment sort, is an in-place comparison-based sorting algorithm. It improves upon the insertion sort algorithm by breaking the original list into smaller sublists and then sorting these sublists using insertion sort. The unique feature of Shell Sort is the way in which the elements are compared and moved.

History

Shell Sort was introduced by Donald Shell in 1959. It was the first sorting algorithm to break the quadratic time barrier. Although not as fast as some other sorting algorithms, it performs significantly better than insertion sort and bubble sort.

Working Principle and Example

Consider an array to be sorted: [12, 34, 54, 2, 3].

Step.1
  • Choose a gap sequence (e.g., Knuth sequence)
  • For the given example, let's choose a gap sequence: 4, 2, 1
Step.2
  • Iterate over the elements of the array for each gap size
  • For the given example:
    • For gap size 4: [3, 2, 54, 12, 34]
    • For gap size 2: [3, 2, 12, 54, 34]
    • For gap size 1: [2, 3, 12, 34, 54]
Result

[2, 3, 12, 34, 54] (sorted)

Time Complexity

  • Time complexity: \(O(n \log^2 n)\)

Summary

Shell Sort is an efficient sorting algorithm that is especially useful for medium-sized lists. It improves upon the insertion sort algorithm by breaking the list into smaller sublists and then sorting these sublists independently using an insertion sort. Despite its slightly higher time complexity than some other algorithms, it is still a practical choice for many sorting tasks.

Java code

        public void sort() {
            int[] list = super.getList();
            int gap = list.length / 2;
    
            while (gap > 0) {
                for (int left = 0; left < gap; left++) {
                    for (int i = left + gap; i < list.length; i += gap) {
                        int target = i;
                        for (int j = i - gap; j >= left; j -= gap) {
                            if (list[target] < list[j])
                                swap(target--, j);
                            else
                                break;    
                        }
                    }    
                }
                gap /= 2;
            }
        }
