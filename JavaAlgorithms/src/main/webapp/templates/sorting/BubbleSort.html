Bubble Sort

Bubble Sort is one of the simplest and most intuitive sorting algorithms. The name comes from the way larger elements "bubble" to the top of the list during the sorting process.

History

Bubble Sort became widely known in the 1960s and is primarily used for educational purposes. While not an efficient algorithm for large datasets, it is useful for teaching the basic principles of sorting algorithms.

Working Principle

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. This process is repeated until the array is sorted. With each iteration, the largest unsorted element is moved to its correct position at the end of the array. Bubble Sort is an in-place sorting algorithm, meaning it requires very little additional memory space, with a space complexity of \(O(1)\). It is also a stable sorting algorithm, meaning that equal elements retain their relative order.

Consider an array to be sorted: [5, 3, 8, 4, 2].

Step.1
  • [5, 3, 8, 4, 2]
  • Compare 5 and 3 → swap → [3, 5, 8, 4, 2]
  • Compare 5 and 8 → no swap
  • Compare 8 and 4 → swap → [3, 5, 4, 8, 2]
  • Compare 8 and 2 → swap → [3, 5, 4, 2, 8]
  • The largest element 8 is now in its correct position.
Step.2
  • [3, 5, 4, 2, 8]
  • Compare 3 and 5 → no swap
  • Compare 5 and 4 → swap → [3, 4, 5, 2, 8]
  • Compare 5 and 2 → swap → [3, 4, 2, 5, 8]
  • The second largest element 5 is now in its correct position.
Step.3
  • [3, 4, 2, 5, 8]
  • Compare 3 and 4 → no swap
  • Compare 4 and 2 → swap → [3, 2, 4, 5, 8]
  • The third largest element 4 is now in its correct position.
Step.4
  • [3, 2, 4, 5, 8]
  • Compare 3 and 2 → swap → [2, 3, 4, 5, 8]
  • All elements are now sorted.
Result

[2, 3, 4, 5, 8] (sorted)

Time Complexity

  • Best-case time complexity: \(O(n)\) (when the array is already sorted)
  • Average and worst-case time complexity: \(O(n^2)\)

Summary

Bubble Sort is a simple and easy-to-understand algorithm, but its efficiency is lacking for large datasets. However, it is very useful for educational purposes to teach the fundamental concepts of sorting algorithms. This information aims to enhance understanding of Bubble Sort and provide a foundational knowledge for comparing it with other sorting algorithms.
